Creation Process

       How did I make the PowerApps?

* Using Microsoft PowerApp software tool 
* I learned how to use PowerApps through video tutorials/trial and error 
* Using Sharepoint data, I connected each e-form to a sharepoint list for data collection 
* Each PowerApp has a seperate microsoft flow equipped with an HTML table that is emailed to supervisors each time an e-form is submitted to the sharepoint site

   How did I make the prototypes +     Mockups?

* I used Balsamiq prototype tool to create the prototype  
* The mockups were created entirely using google slide shapes and tools
* Through the design thinking process I learned to empathize, ideate and prototype for each user these were designed for 
* Collaboration with classmates who brought their own ideas to the prototype design process
